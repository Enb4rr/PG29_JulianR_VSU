# Unreal Simple Shooting Range

A simple **Unreal Engine** project showcasing a basic **shooting range** experience. This project is intended as a small gameplay prototype demonstrating core systems such as player control, interaction, UI, and audio integration.

---

## Controls

* **W / A / S / D** – Move character
* **Left Mouse Button** – Shoot
* **E** – Interact

---

## Implemented Systems & Components

This project includes the following implemented components:

* **Menus** – Basic navigation and game flow menus
* **User Interfaces (UI)** – HUD and interactive UI elements
* **Sounds** – Shooting, interaction, and feedback audio
* **Movement System** – Player locomotion and camera control
* **Managers** – Centralized logic controllers (game/state handling)
* **Inheritance** – Base classes and child classes for reusable gameplay logic
